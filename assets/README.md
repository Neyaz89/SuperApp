# Assets Directory

This directory contains app assets including icons and splash screens.

## Required Assets

### App Icon (icon.png)
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Design: App logo on transparent or solid background

### Adaptive Icon (adaptive-icon.png)
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Design: App logo centered, safe area 512x512

### Splash Screen (splash.png)
- Size: 1284x2778 pixels (or larger)
- Format: PNG
- Design: App logo centered on black background

### Favicon (favicon.png)
- Size: 48x48 pixels
- Format: PNG
- Design: Simplified app icon

## Generating Assets

You can use the following tools to generate assets:
- Figma
- Adobe Illustrator
- Canva
- Online icon generators

## Asset Guidelines

1. **Icon Design**
   - Simple and recognizable
   - Works well at small sizes
   - Consistent with brand identity
   - No text (unless it's a logo)

2. **Color Scheme**
   - Primary: #007AFF (iOS Blue)
   - Background: #000000 (Black)
   - Accent colors as needed

3. **Splash Screen**
   - Minimal design
   - Fast loading perception
   - Consistent with app theme

## Placeholder Assets

For development, you can use placeholder images:
- icon.png: 1024x1024 solid color with app initial
- splash.png: 1284x2778 black background with centered logo
- adaptive-icon.png: Same as icon.png
- favicon.png: 48x48 simplified icon
