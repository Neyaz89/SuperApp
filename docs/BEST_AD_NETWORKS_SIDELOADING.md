## Unity Ads Setup Guide

### Step 1: Register

1. Go to https://unity.com/products/unity-ads
2. Click "Get Started"
3. Create Unity account (free)
4. No credit card required

### Step 2: Create Project

1. Login to Unity Dashboard
2. Create new project: "SuperHub"
3. Platform: Android
4. Distribution: Other

### Step 3: Get IDs

You'll receive:
- **Game ID:** 1234567 (example)
- **Placement IDs:**
  - Banner: "banner_android"
  - Interstitial: "interstitial_android"
  - Rewarded: "rewarded_android"

### Step 4: Integration

```bash
npm install @unity-ads/react-native
```

**Configuration:**
```typescript
import UnityAds from '@unity-ads/react-native';

// Initialize
UnityAds.initialize('1234567', false); // false = production

// Show banner
UnityAds.showBanner('banner_android');

// Show interstitial
UnityAds.showInterstitial('interstitial_android');

// Show rewarded
UnityAds.showRewarded('rewarded_android');
```

### Optimization Tips:

1. **Ad Frequency:**
   - Banner: Always visible
   - Interstitial: Every 2-3 actions
   - Rewarded: Before extraction

2. **Placement Strategy:**
   - High-value actions = Rewarded
   - Natural breaks = Interstitial
   - Persistent = Banner

3. **User Experience:**
   - Don't spam ads
   - Reward users properly
   - Smooth transitions
